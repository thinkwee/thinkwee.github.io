数据概览

  • describe 显示整体数据常见属性
  • head tail 显示首尾一些数据
  • index：索引，默认自建整型索引；columns：列；values：数据数值 # 数据操作
  • T：数据的转置
  • sort：可以按索引或者值进行排序，axis选择维度(行还是列),ascending选择升序或者降序,Nan永远排在最后，无论升序降序

数据选择

  • 按照标签选择，选择列，行切片
  • 利用loc自由选择某些行某些列，可以用at替代
  • 利用iloc按照位置进行选择
  • 布尔选择

缺失值处理

  • 利用reindex选择部分数据进行拷贝，并进行缺失值处理。一些函数会自动过滤掉缺失值，比如mean() # 应用函数
  • 可以自己写函数并应用到数据的行或者列，通过axis参数选择行列 # 数据透视表
  • 自选分类和值进行数据透视，比如按照pclass和sex分类，统计age和fare的平均值
