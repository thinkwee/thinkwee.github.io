  • жӢүж јжң—ж—Ҙд№ҳеӯҗжі•жҳҜдёҖз§ҚжұӮзәҰжқҹжқЎд»¶дёӢжһҒеҖјзҡ„ж–№жі•пјҢжҸҸиҝ°дёәе…¶дё»иҰҒжҖқжғіжҳҜе°ҶзәҰжқҹжқЎд»¶е’ҢеҺҹеҮҪж•°еҗҲжҲҗдёҖдёӘеҮҪж•°пјҢиҪ¬жҚўдёәж— зәҰжқҹжқЎд»¶пјҢиҝӣиҖҢжұӮеҒҸеҜјеҫ—еҲ°жһҒеҖјгҖӮ
  • з”ұеӣҫеҸҜд»ҘзңӢеҮәпјҢfеҮҪж•°еҖјзӣёзӯүзҡ„зӮ№еҸҜд»Ҙжһ„жҲҗзұ»дјјзӯүй«ҳзәҝзҡ„и“қиүІзҺҜпјҢзәҰжқҹжқЎд»¶жҳҜз»ҝиүІзҡ„и·Ҝеҫ„гҖӮй—®йўҳеҸҜд»ҘиҪ¬жҚўдёәпјҢжҲ‘д»¬жІҝзқҖз»ҝиүІи·Ҝеҫ„иө°пјҢиө°еҲ°е“ӘдёҖзӮ№ж—¶иҝҷдёӘзӮ№жүҖеңЁзҡ„и“қиүІзҺҜжңҖйқ дёӯеҝғжҲ–иҖ…жңҖйқ еӨ–жІҝ(жһҒеӨ§жһҒе°ҸеҖј)гҖӮ
  • жҳҫз„¶пјҢеңЁз»ҝиүІи·Ҝеҫ„дёҺи“қиүІзҺҜзӣёеҲҮзҡ„зӮ№еҸ–еҫ—жһҒеҖјпјҢжӯӨж—¶е®ғд»¬зҡ„жўҜеәҰ(з®ӯеӨҙ)е№іиЎҢпјҢжҸҸиҝ°дёә$\lambda$жҳҜжӢүж јжң—ж—Ҙд№ҳж•°пјҢеңЁиҝҷдёӘејҸеӯҗдёӯд»ЈиЎЁдёӨдёӘе№іиЎҢжўҜеәҰзҡ„еӨ§е°ҸеҖҚж•°пјҢжӯЈиҙҹд»ЈиЎЁдёӨдёӘжўҜеәҰж–№еҗ‘зӣёеҸҚгҖӮжӢүж јжң—ж—Ҙд№ҳж•°дёҚдёә0гҖӮ
    жӢүж јжң—ж—Ҙж–№зЁӢеҚі$ F(x,y)=\nabla \Big[f \left(x, y \right) + \lambda \left(g \left(x, y \right) - c \right) \Big] $
  • жұӮи§ЈдёҠйқўзҡ„ејҸеӯҗпјҢе°ұеҫ—еҲ°дёҖз»„$(x,y,\lambda)$пјҢеҚіжһҒеҖјзӮ№е’ҢиҫҫеҲ°жһҒеҖјж—¶зҡ„жӢүж јжң—ж—Ҙд№ҳж•°гҖӮжӯӨж—¶жӢүж јжң—ж—Ҙж–№зЁӢ$F(x,y)=f(x,y)$пјҢеӣ дёәеҸ–еҫ—жһҒеҖјж—¶зәҰжқҹжқЎд»¶йғЁеҲҶдёҖе®ҡдёә0(жҲ‘д»¬жҳҜжІҝзқҖзәҰжқҹжқЎд»¶иө°жүҫзӣёеҲҮзӮ№пјҢзӣёеҲҮзӮ№еңЁзәҰжқҹи·Ҝеҫ„дёҠ)гҖӮ
  • жӢүж јжң—ж—Ҙзі»ж•°зҡ„еҗ«д№үжҳҜжңҖеӨ§еўһй•ҝеҖјпјҢ$-\frac{\partial \Lambda}{\partial {c_k}} = \lambda_k$

еҚЎзҪ—йңҖ-еә“жҒ©-еЎ”е…ӢжқЎд»¶

  • еҰӮжһңзәҰжқҹжқЎд»¶дёҚд»…д»…жҳҜзӯүејҸпјҢиҝҳеҢ…жӢ¬дёҚзӯүзәҰжқҹжқЎд»¶пјҢиҝҷе°ұйңҖе°ҶжӢүж јжң—ж—Ҙд№ҳеӯҗжі•жҺЁе№ҝеҲ°KKTжқЎд»¶
  • еҢ…еҗ«дёҚзӯүзәҰжқҹзҡ„жһҒеҖјй—®йўҳжҸҸиҝ°дёә

PCA

  • PCAеҚіPrincipal Component AnalysisпјҢдё»жҲҗеҲҶеҲҶжһҗпјҢе°Ҷж•°жҚ®еҺҹжң¬зҡ„з»ҙеәҰиҝӣиЎҢдјҳеҢ–пјҢеҪўжҲҗдёҖз»„ж–°зҡ„з»ҙеәҰпјҢе®ғд»¬жҳҜеҺҹжңүз»ҙеәҰзҡ„зәҝжҖ§з»„еҗҲдё”дә’дёҚзӣёе…іпјҢжҢүйҮҚиҰҒжҖ§еӨ§е°ҸжҺ’еәҸпјҢдёҖдёӘз»ҙеәҰеҸҜд»ҘзҗҶи§Јдёәж•°жҚ®зҹ©йҳөдёӯдёҖеҲ—пјҢдёҖиЎҢд»ЈиЎЁдёҖдёӘж ·жң¬
  • и®°$x_1,вҖҰ,x_p$дёәеҺҹе§ӢpдёӘз»ҙеәҰпјҢж–°з»ҙеәҰжҳҜ$\xi _1,вҖҰ.,\xi _p$
  • ж–°з»ҙеәҰжҳҜеҺҹе§Ӣз»ҙеәҰзҡ„зәҝжҖ§з»„еҗҲпјҢиЎЁзӨәдёә
  • дёәдәҶеҗ„дёӘж–°з»ҙеәҰз»ҹдёҖе°әеәҰпјҢеҸҰжҜҸдёӘж–°з»ҙеәҰзҡ„зәҝжҖ§з»„еҗҲзі»ж•°зҡ„еҗ‘йҮҸй•ҝеәҰйғҪдёә1пјҢеҚі
  • д»ӨAдёәзү№еҫҒеҸҳжҚўзҹ©йҳөпјҢз”ұеҗ„дёӘж–°з»ҙеәҰзҡ„зәҝжҖ§з»„еҗҲзі»ж•°еҗ‘йҮҸжһ„жҲҗпјҢеҲҷйңҖиҰҒжұӮи§ЈдёҖдёӘжңҖдјҳзҡ„жӯЈдәӨеҸҳжҚўAпјҢдҪҝеҫ—ж–°з»ҙеәҰзҡ„ж–№е·®иҫҫеҲ°жһҒеҖјгҖӮе…¶дёӯжӯЈдәӨеҸҳжҚўеҚідҝқиҜҒеҗ„дёӘж–°з»ҙеәҰдёҚзӣёе…іпјҢж–№е·®и¶ҠеӨ§еҲҷж ·жң¬еңЁж–°з»ҙеәҰдёҠе…·жңүеҢәеҲҶеәҰпјҢж–№дҫҝжҲ‘д»¬иҝӣиЎҢж•°жҚ®зҡ„еҲҶзұ»
  • жӯӨж—¶й—®йўҳе°ұиҪ¬еҢ–дёәе…·жңүзәҰжқҹжқЎд»¶зҡ„жһҒеҖјй—®йўҳпјҢзәҰжқҹжқЎд»¶дёә$\alpha _i^T \alpha _i=1$пјҢжұӮ$var(\xi _i)$зҡ„жһҒеҖјпјҢеҸҜд»Ҙз”ЁжӢүж јжң—ж—Ҙд№ҳеӯҗжі•жұӮи§Ј
  • еҪ“i=1ж—¶пјҢжҲ‘д»¬жұӮеҮәжқҘз¬¬дёҖдёӘд№ҹжҳҜйҮҚиҰҒжҖ§жңҖеӨ§(ж–№е·®жңҖеӨ§)зҡ„ж–°з»ҙеәҰпјҢеҶҚд»Өi=2,е№¶еҠ е…ҘзәҰжқҹжқЎд»¶$E[\xi _2 \xi _1-E[\xi _1][\xi _2]]=0$еҚідёӨдёӘж–°з»ҙеәҰдёҚзӣёе…іпјҢжұӮеҮәз¬¬дәҢдёӘж–°з»ҙеәҰ
  • дҫқж¬ЎжұӮеҮәpдёӘж–°з»ҙеәҰ
  • PCAиғҪеӨҹдјҳеҢ–еҺҹе§Ӣж•°жҚ®пјҢжүҫеҮәе…·жңүеҢәеҲҶеәҰзҡ„з»ҙеәҰпјҢжӣҙйҮҚиҰҒзҡ„жҳҜеҰӮжһңеҺҹе§Ӣж•°жҚ®зҡ„з»ҙеәҰеӯҳеңЁзӣёе…іжҖ§пјҢPCAиғҪж¶ҲйҷӨиҝҷдәӣзӣёе…іжҖ§пјҢеҚідҫҝеҺҹе§Ӣж•°жҚ®зӣёе…іжҖ§еҫҲдҪҺпјҢеҰӮжһңжҲ‘д»¬еҸӘеҸ–еүҚk(k<q)дёӘж–°з»ҙеәҰпјҢе°ұеҸҜд»ҘеңЁжҚҹеӨұиҫғе°ҸзІҫзЎ®еәҰзҡ„жғ…еҶөдёӢиҝӣиЎҢйҷҚз»ҙпјҢеӨ§еӨ§зј©зҹӯж•°жҚ®зҡ„и®ӯз»ғж—¶й—ҙ
  • еҰӮжһңжҲ‘д»¬еҸ–дәҶеүҚkдёӘж–°з»ҙеәҰпјҢеҶҚеҜ№д»–д»¬иҝӣиЎҢPCAзҡ„йҖҶиҝҗз®—пјҢе°ұеҸҜд»Ҙе®һзҺ°ж•°жҚ®зҡ„йҷҚеҷӘпјҢеӣ дёәйҮҚиҰҒжҖ§еҫҲдҪҺзҡ„ж–°з»ҙеәҰдёҖиҲ¬еҸҚеә”дәҶж•°жҚ®дёӯзҡ„йҡҸжңәеҷӘеЈ°пјҢжҠӣејғе®ғд»¬е№¶жҒўеӨҚеҺҹе§Ӣж•°жҚ®ж—¶е°ұе®һзҺ°дәҶеҷӘйҹізҡ„еҺ»йҷӨ
  • PCAжҳҜйқһзӣ‘зқЈзҡ„пјҢжІЎжңүиҖғиҷ‘ж ·жң¬жң¬иә«зҡ„зұ»еҲ«жҲ–иҖ…ж ҮзӯҫпјҢеңЁзӣ‘зқЈеӯҰд№ дёӯдёҚдёҖе®ҡжҳҜжңҖдјҳи§ЈпјҢеҸҜд»ҘеҲ©з”ЁK-LеҸҳжҚўе®һзҺ°й’ҲеҜ№еҲҶзұ»зҡ„зӣ®ж ҮиҝӣиЎҢзү№еҫҒжҸҗеҸ–

PCA using Covariance Matrix

  • дёҠиҝ°жұӮи§ЈPCAзҡ„ж–№жі•еӨӘиҝҮдәҺйә»зғҰпјҢеңЁе®һйҷ…дёӯеҸҜд»ҘйҖҡиҝҮеҚҸж–№е·®зҹ©йҳөжқҘжұӮи§ЈпјҲеӣ дёәжңүй«ҳж•Ҳзҡ„зҹ©йҳөзү№еҫҒеҲҶи§Јз®—жі•пјү
  • PCAзҡ„дјҳеҢ–зӣ®ж ҮжҳҜпјҢйҮҚж–°йҖүеҸ–дёҖз»„еҹәпјҲзү№еҫҒпјүпјҢдҪҝеҫ—ж•°жҚ®еңЁиҝҷз»„еҹәиЎЁзӨәдёӢпјҢдёҚеҗҢзү№еҫҒд№Ӣй—ҙзҡ„еҚҸж–№е·®дёә0пјҢеҗҢдёҖзү№еҫҒеҶ…зҡ„ж•°жҚ®ж–№е·®жңҖеӨ§еҢ–
  • еҸҜд»Ҙе°Ҷй—®йўҳиҪ¬иҝ°дёәпјҢеҜ№ж•°жҚ®еј йҮҸXпјҢжҢүзү№еҫҒеҲ—пјҲе…ұmдёӘзү№еҫҒпјүйӣ¶еқҮеҖјеҢ–пјҲеҢ–з®ҖеҚҸж–№е·®зҡ„и®Ўз®—пјүпјҢи®Ўз®—еҚҸж–№е·®зҹ©йҳө$C=\frac 1m X^T X$пјҢеёҢжңӣжұӮеҫ—дёҖз»„еҹәPпјҢдҪҝеҫ—зү№еҫҒеҸҳжҚўеҗҺзҡ„ж•°жҚ®$Y=PX$зҡ„еҚҸж–№е·®зҹ©йҳөDжҳҜеҜ№и§’йҳөпјҢйқһеҜ№и§’е…ғзҙ пјҲеҚҸж–№е·®пјүдёә0.иӢҘеҜ№и§’е…ғзҙ пјҲж–№е·®пјүжҢүд»ҺеӨ§еҲ°е°ҸжҺ’еҲ—пјҢиҝҷж—¶жҲ‘д»¬еҸ–Pзҹ©йҳөзҡ„еүҚkиЎҢе°ұеҸҜд»Ҙе°Ҷзү№еҫҒз»ҙеәҰд»ҺmйҷҚеҲ°kгҖӮжҳ“еҫ—$D=PCP^T$пјҢдё”Cдёәе®һеҜ№з§°йҳөпјҢйӮЈд№Ҳй—®йўҳе°ұиҪ¬еҸҳдёәеҜ№е®һеҜ№з§°CеҜ№и§’еҢ–пјҢжҲ‘д»¬йңҖиҰҒзҡ„ж–°зҡ„дёҖз»„еҹәе°ұжҳҜзү№еҫҒеҗ‘йҮҸз»„гҖӮ
  • з®—жі•пјҡ
    • жңүmжқЎnз»ҙж•°жҚ®пјҢжҺ’жҲҗnиЎҢmеҲ—зҹ©йҳөX
    • е°ҶXзҡ„жҜҸдёҖиЎҢиҝӣиЎҢйӣ¶еқҮеҖјеҢ–
    • жұӮеҮәеҚҸж–№е·®зҹ©йҳөC
    • жұӮеҮәеҚҸж–№е·®зҹ©йҳөзҡ„зү№еҫҒеҖје’Ңзү№еҫҒеҗ‘йҮҸ
    • е°Ҷзү№еҫҒеҗ‘йҮҸжҢүзү№еҫҒеҖјеӨ§е°ҸеҜ№еә”д»ҺеӨ§еҲ°е°ҸжҢүиЎҢжҺ’еҲ—пјҢз»„жҲҗж–°зҡ„еҹәзҹ©йҳөP
    • еҰӮжһңйңҖиҰҒе°ҶдёәпјҢеҸ–Pзҡ„еүҚkиЎҢеҚіеҸҜпјҢйҷҚз»ҙеҗҺзҡ„ж•°жҚ®дёә$Y=PX$

SVM

  • еңЁеҲҶзұ»еӯҰд№ дёӯпјҢжҲ‘д»¬йңҖиҰҒжүҫеҲ°дёҖдёӘеҲ’еҲҶи¶…е№ійқўпјҢе°ҶдёҚеҗҢзұ»еҲ«зҡ„ж ·жң¬еҲҶејҖпјҢиҖҢжңҖеҘҪзҡ„еҲ’еҲҶи¶…е№ійқўжҳҫз„¶жҳҜзҰ»жүҖеҲҶеҗ„дёӘж ·жң¬зұ»е°ҪйҮҸиҝңпјҢеҚіеҜ№и®ӯз»ғж ·жң¬еұҖйғЁжү°еҠЁе®№еҝҚжҖ§жңҖеҘҪзҡ„и¶…е№ійқў
  • еҲ’еҲҶи¶…е№ійқўйҖҡиҝҮж–№зЁӢ$w^Tx+b=0$жҸҸиҝ°пјҢе…¶дёӯwдёәжі•еҗ‘йҮҸпјҢеҶіе®ҡдәҶи¶…е№ійқўж–№еҗ‘пјҢbдёәи¶…е№ійқўеҲ°иҝңзӮ№зҡ„дҪҚз§»
  • жұӮи§ЈдёҖдёӘSVMпјҢеҚіжүҫеҲ°ж»Ўи¶ізәҰжқҹзҡ„жқЎд»¶дёӢпјҢдҪҝеҫ—дёӨдёӘејӮзұ»ж”ҜжҢҒеҗ‘йҮҸеҲ°и¶…е№ійқўзҡ„и·қзҰ»$\frac{2}{||w||}$жңҖеӨ§
    иҝҷеҸҜд»ҘйҮҚеҶҷдёәжңҖдјҳеҢ–й—®йўҳжҺЁеҜји§ҒеҸҰдёӨзҜҮеҚҡж–ҮпјҡжңәеҷЁеӯҰд№ з¬”и®°е’Ңз»ҹи®ЎеӯҰд№ ж–№жі•з¬”и®°жүӢеҶҷзүҲ
  • еҜ№дәҺиҝҷдёӘжңҖдјҳеҢ–й—®йўҳпјҢе®ғзҡ„жӢүж јжң—ж—Ҙж–№зЁӢжҳҜе…¶дёӯ$\alpha$жҳҜжӢүж јжң—ж—Ҙд№ҳеӯҗпјҢд»Өж–№зЁӢеҲҶеҲ«еҜ№w,bжұӮеҒҸеҜјпјҢеҫ—еҲ°еҜ№еҒ¶й—®йўҳдёҠејҸж»Ўи¶іKKTжқЎд»¶пјҢйҖҡиҝҮSMOз®—жі•жұӮи§Ј
